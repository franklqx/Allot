# Oura Ring App

[awesome-design-md](https://github.com/VoltAgent/awesome-design-md) **这个仓库里确实有「合集」**：README 里的 **Collection** 是一长串品牌，并且每个品牌在 GitHub 上有 `design-md/<名字>/` 目录。

但要注意两件事：

1. **目录里通常没有完整的 `DESIGN.md` 文件**，往往只有一个 `README.md`，里面写「正文已迁移到 getdesign.md」，真实正文靠 `npx getdesign add <品牌>` 拉取。
2. **品牌名里没有 Oura**：我核对过当前 `main` 分支下 `design-md/` 子目录列表与 README 全文，**没有** `oura`、`oura-ring` 或类似条目；`npx getdesign list` 里同样**没有** Oura。

所以：**合集在，Oura 这一条不在合集里。**

**你可以：**

- 到 [getdesign.md/request](https://getdesign.md/request) 申请收录 Oura。
- 或自建 `DESIGN.md`（章节结构可参考 `../apple-health/DESIGN.md`），并对照 `PRODUCT_SPEC.md` §13.4 里对 Oura 的描述。
